# How it works

DeFi Intents is an aggregator API that unifies actions like swaps and bridges behind one consistent interface.

## What you do

1. Send an intent request from your app (your parameters).
2. Receive a structured result (your integration output).

## What DeFi Intents does

1. Understand the requested DeFi action across providers.
2. Select the right capabilities and execute the flow.
